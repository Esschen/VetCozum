Vaccinations

Important Note: All vaccines must be administered and consulted by a veterinarian.

Core Vaccines For Dogs:

Canine parvovirus, Distemper, Canine hepatitis, Rabies

Core Vaccines For Cats:

Feline panleukopenia (distemper), Feline viral rhinotracheitis, Feline calicivirus, Rabies

Core Vaccines For Beef Cattles:

IBR, BVD, BRSV, PI3 *Must use killed vaccine if cows are pregnant.

Core Vaccines For Dairy Calves:

IBR, PI3, BVD, BRSV-MLV

Core Vaccines For Horses:

Strangles, encephalitis, rhino, flu, west nile, tetanus, worm
